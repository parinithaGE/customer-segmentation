# Customer Segmentation Using Machine Learning

## Project Overview
This project uses machine learning techniques to segment customers into different groups based on their purchasing behavior and demographics. Customer segmentation helps businesses understand their customers better and create targeted marketing strategies.

## Objectives
- Analyze customer data
- Identify customer groups with similar characteristics
- Visualize customer segments
- Support business decision-making

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Dataset
The dataset contains customer information such as:
- Customer ID
- Age
- Gender
- Annual Income
- Spending Score

## Methodology
1. Data Preprocessing
2. Exploratory Data Analysis (EDA)
3. Feature Selection
4. K-Means Clustering
5. Visualization of Customer Segments

## Results
The model successfully grouped customers into distinct segments based on their spending patterns and income levels.
